# interviewBible
面试宝典刷题小程序
* 类别的增删改查
* 题目的增删改查
* html页面直接跳转传值问题
* layedit富文本框的应用，支持对齐方式、字体粗细、下划线、删除线等样式调整
* spring security的浅应用
